Project Details

참좋은 여행 홈페이지 리모델링

  • 작업 유형: 팀 프로젝트 (기획 담당 및 제작 보조)
  • 개발 언어: HTML5, CSS, JQuery, Fontawesome, Google Fonts, PHP, MySQL
  • 개발 도구: Figma, VS Code
  • 프로젝트명: 참좋은 여행 홈페이지 리모델링
  • 제작 기간: 2 Weeks
  • 사이트 유형: 반응형 (PC, Tablet, Mobile)
  • 해상도: 1440px

Preview

메인페이지

  • - 메인 슬라이드
  • - 헤더 검색 기능
  • - 랜덤 배너 광고
  • - 여행 상품 리스트 가로 스크롤로 제공

카테고리 메뉴

  • - 카테고리 클릭 시 해당하는 여행 상품 목록 제공
  • - 원본 참좋은 여행 사이트에서 나뉘어져 있던 카테고리별 상품들을 합쳐서 하나로 깔끔하게 정리

관리자 페이지

  • - 로그인 페이지에서 관리자 계정으로 로그인시 관리자 메인 페이지로 이동
  • - 회원, 여행지 관리, 여행 일정 관리, 고객센터 관리 기능
  • - 로그아웃시 참좋은 여행의 고객 메인 페이지로 이동

상품 관리

  • - 상품 관리 및 수정 클릭 시 여행 상품 목록 페이지로 이동
  • - 상품 이름 클릭 시/상품 수정 완료시 상품 상세 페이지로 이동
  • - 상품 상세 페이지 하단에 해당 상품의 일정 목록 표시
  • - 상품의 일정 목록에서 해당 상품 일정 수정 가능
